# Scalable Online Product Management System (AWS)

## 📌 Project Overview
A real-time scalable 3-tier web application deployed on AWS using Elastic Beanstalk, EC2 Auto Scaling, Application Load Balancer, and Amazon RDS (MySQL).

## 🏗 Architecture
- Presentation Tier: Browser
- Application Tier: Elastic Beanstalk (Flask on EC2)
- Database Tier: Amazon RDS (MySQL)

## ⚙️ AWS Services Used
- Elastic Beanstalk
- EC2 & Auto Scaling
- Application Load Balancer
- Amazon RDS (MySQL)
- IAM
- CloudWatch

## 🚀 Features
- Scalable deployment
- Load balanced traffic
- Secure DB connectivity using Security Groups
- Environment-based configuration
- Health monitoring & auto recovery
